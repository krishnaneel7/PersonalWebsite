IoT Based Wireless EV Charging System

Project Overview

This project focuses on developing an IoT-enabled wireless charging system for electric vehicles.

Hardware Components

  • ESP8266 NodeMCU
  • RFID RC522 Module
  • Relay Module
  • Voltage Sensor
  • Blynk IoT Platform
  • Wireless Transmitter and Receiver Coil

Working Principle

The RFID reader authenticates the user. After successful authentication, the relay activates the transmitter coil. Wireless power is transferred through electromagnetic induction to the receiver coil. The ESP8266 monitors battery voltage and sends data to the Blynk IoT platform.
